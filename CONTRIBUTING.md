# Contributing

Thank you for contributing to TransportVault.

## Project Goals

TransportVault is intended to provide:

- Secure image storage
- Golden image deployment
- Recovery workflows
- Auditability
- Compliance-oriented architecture

## Pull Request Requirements

- Clear description
- Test evidence
- Documentation updates
- Backwards compatibility assessment

## Code Quality

- Python 3.11+
- Type hints preferred
- Modular design
- Minimal dependencies

## Security Review

Changes affecting the following require additional review:

- Encryption
- Vault management
- Authentication
- Audit logging
- Verification
- Compliance features

## Compliance Language

Avoid unsupported claims regarding:

- FIPS certification
- NIST compliance
- RMF authorization
- STIG compliance

Such claims must be supported by documentation and evidence.
